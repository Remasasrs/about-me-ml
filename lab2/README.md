Lab 2 – Machine Learning Problem Definition

Problem Description:
This project aims to predict a student’s final exam score based on the number of hours studied. This is a supervised machine learning regression problem, as the target variable is a continuous numerical value.

Dataset Description:
The dataset contains student-related data such as study hours and academic performance. It is a tabular dataset in CSV format and is suitable for regression analysis. The target variable in this dataset is the student’s final exam score.
